Acceso administrador

Ingresá la contraseña para actualizar el catálogo de precios.

Panel de administración

1. Descargar el catálogo actual

Bajá el Excel con los precios que están hoy publicados en la web para editarlo.

Descargar Excel actual

2. Subir el Excel actualizado

Editá los precios en el archivo descargado (no cambies los nombres de las columnas) y subilo acá.

3. Imágenes de productos

Buscá un producto y subí o reemplazá su foto. Se guarda como miniatura optimizada automáticamente.

    4. Revisión de imágenes automáticas

    Estas fotos se completaron automáticamente buscando en internet por nombre y marca. No son fotos reales del producto: muchas están mal (objetos, personas o escenas que no tienen nada que ver). Marcá las que estén mal y descartalas — el producto queda sin foto hasta que subas la correcta a mano (sección 3) o corras de nuevo el script de búsqueda.

    Formato del Excel

    Podés subir directamente el Excel tal cual lo exporta el sistema interno (la lista agrupada por familia, igual a la lista de precios de siempre) — se reconoce automáticamente, sin necesidad de reformatear nada.

    También se acepta una planilla simple con estas columnas (en cualquier orden, con estos nombres exactos; ImagenURL es opcional):

    CategoriaProductoPrecioImagenURL
    ResmasResma A4 Chamex x 500 hojas5500
    EscrituraBolígrafo Bic Cristal Azul350,50https://...

    Las filas sin categoría, sin producto o con un precio inválido se descartan automáticamente y se avisa cuáles fueron. Las fotos que subiste desde la sección de arriba no se pierden al actualizar precios, salvo que el Excel traiga una ImagenURL nueva para ese producto.
